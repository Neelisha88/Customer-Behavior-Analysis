\# Customer Behavior Data Analysis Project



This project demonstrates a complete end-to-end data analytics workflow using \*\*Python, SQL, and Power BI\*\*. The objective is to analyze customer shopping behavior and extract insights that help understand purchasing trends, customer segments, and product performance.



The project simulates a real-world business scenario where raw data is transformed into meaningful insights through data cleaning, analysis, and visualization.

This project was developed as part of hands-on practice to strengthen practical data analytics skills and simulate real-world business analysis scenarios.

---



\# Project Objectives



The main goals of this project are:



\* Understand customer purchasing behavior

\* Analyze product and category performance

\* Identify sales trends and patterns

\* Generate insights that support data-driven business decisions

\* Create an interactive dashboard for visualization

\# Project Workflow



\### 1. Data Preparation (Python)



The dataset was imported and cleaned using Python. Steps included:



\* Handling missing values

\* Data formatting and transformation

\* Initial exploratory analysis



---



\### 2. Data Analysis (SQL)



The cleaned dataset was loaded into a SQL database and analyzed using SQL queries to answer business questions such as:



\* Which products generate the highest revenue?

\* Which categories perform best?

\* How customer purchasing behavior varies



---



\### 3. Data Visualization (Power BI)



An interactive Power BI dashboard was created to visualize the analysis results. The dashboard includes:



\* Sales overview metrics

\* Product category performance

\* Customer behavior insights

\* Trend analysis



---



\# Key Insights



Some insights derived from the analysis include:



\* A small number of product categories contribute significantly to total revenue.

\* Customer purchasing patterns vary across product types.

\* Certain products consistently generate higher demand.
